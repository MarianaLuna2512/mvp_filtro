# mvp_filtro
# Tu artista espiritual — Propuesta de proyecto

**Tipo:** Prueba de concepto (PoC) · Proyecto académico con potencial comercial  
**Desarrollador:** 1 persona · Menos de 2 semanas

---

## Descripción del proyecto

Un archivo HTML único que el usuario abre en el navegador. Sin instalación, sin npm, sin deploy obligatorio. El usuario responde 5 preguntas sobre sus gustos musicales, emociones y experiencias, y el sistema revela cuál artista musical real —Bad Bunny, Karol G, Shakira, entre otros— es su "artista espiritual": aquel cuya identidad artística conecta con su forma de sentir y vivir la música.

El resultado se presenta como una tarjeta visual con el nombre del artista, una descripción del match y canciones representativas.

---

## Problema que resuelve

Las plataformas musicales tienen acceso a datos de reproducción, pero no capturan la identidad emocional del usuario: cómo se siente con la música, qué experiencias la rodean, qué lo mueve más allá de un género. Este proyecto convierte esa identidad difusa en un resultado concreto, memorable y compartible — un tipo de dato que plataformas como Spotify o YouTube Music hoy no pueden obtener fácilmente.

**Argumento central para plataformas:** este filtro no es solo entretenimiento — es un motor de perfilamiento emocional disfrazado de juego. Cada respuesta es un dato de intención musical que el streaming convencional no puede capturar.
