# mvp_filtro
Es un proyecto MVP para desarrollar un filtro en el cual la persona tiene que responder ciertas preguntas para así poder indentificar el su "artista espiritual", basandose en los gustos y preguntas de experiencia que las persona responda
IDEA 
Desarrrollar un filtro inrteractivo con cámara donde los usuarios puedan vivir una experiencia con su "Artista Espiritual", que se realice algunas preguntas sobre el gusto, emociones y experiencias de cada usuario para así poder determinar que artista es el elegido según la categoria. Este filtro convierte esa identidad difusa en un resultado memorable, entrenenido y compartible. 

 Un sistema de filtrado basado en respuestas del usuario, identifique y presente de forma personalizada un artista musical perfecto para plaformas musicales como SPOTIFY y Youtube para llamar más audiencia y segmentar a sus usuarios. 

 Las plataformas musicales tienen acceso a datos de reproducción, pero no capturan la identidad emocional del usuario — cómo se siente con la música, qué experiencias la rodean, qué lo mueve más allá de un género. Este proyecto convierte esa identidad difusa en un resultado concreto, memorable y compartible, generando un tipo de dato que las plataformas hoy no pueden obtener fácilmente.
